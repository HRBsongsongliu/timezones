# Timezones
A simple timezone dataset with countries, country codes, UTCOffset and timezonesIds.

Example of data point: 
` {
	countryCode: "AF",
  countryName: "Afghanistan",
  timeZones: [
  	"Asia/Kabul"
  ],
  UTCOffset: [
  	"UTC +04:30"
    ]
} `

    
The timeZones and UTCOffset have the same index in the array.
